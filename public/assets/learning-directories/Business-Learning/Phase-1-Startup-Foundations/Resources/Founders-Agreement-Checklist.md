# Founders Agreement Checklist

## Purpose
This checklist ensures you cover all critical points when creating a Founders Agreement with your co-founder. Use this as a guide when working with a lawyer.

## Timeline & Next Steps

### Week 1: Discussion
- [ ] Review this checklist together
- [ ] Discuss each section honestly
- [ ] Identify any disagreements
- [ ] Reach tentative agreement on all points

### Week 2: Legal
- [ ] Find startup lawyer (referrals, Cooley GO, Clerky)
- [ ] Schedule consultation
- [ ] Provide lawyer with this checklist (filled out)
- [ ] Review lawyer's questions

### Week 3: Drafting
- [ ] Lawyer drafts Founders Agreement
- [ ] Both review thoroughly
- [ ] Ask questions (no stupid questions!)
- [ ] Request any changes

### Week 4: Signing
- [ ] Both sign Founders Agreement
- [ ] Notarize if required
- [ ] Each founder gets copy
- [ ] Store securely (digital + physical)

## Red Flags üö©

Stop and reconsider if:
- [ ] Co-founder doesn't want to discuss equity or pushes back hard on vesting
- [ ] Co-founder resists putting anything in writing ("We trust each other!")
- [ ] Co-founder wants 50/50 despite unequal contribution
- [ ] Major disagreements that can't be resolved
- [ ] Feeling pressured or rushed
- [ ] Co-founder dismisses importance of legal documentation

**If any red flags**: Pause and address before signing anything.
