# Phase 5 Resources

This folder contains practical templates and tools to help you achieve product-market fit.

## Templates Available

### 1. PMF Survey Template
Complete survey template for measuring product-market fit using the Sean Ellis test.

### 2. Customer Interview Guide
Step-by-step script and questions for conducting effective customer development interviews.

### 3. Feature Prioritization Matrix
Spreadsheet template for prioritizing features using RICE, ICE, and other frameworks.

### 4. Retention Analysis Template
Tools for tracking and analyzing customer retention, cohorts, and churn.

## How to Use These Resources

1. **Start with Customer Interviews** - Use the interview guide to talk to 30+ potential customers
2. **Prioritize Features** - Use the prioritization matrix to decide what to build first
3. **Measure PMF** - Use the PMF survey after customers have used your product for 2-4 weeks
4. **Track Retention** - Use the retention analysis template to monitor your progress
