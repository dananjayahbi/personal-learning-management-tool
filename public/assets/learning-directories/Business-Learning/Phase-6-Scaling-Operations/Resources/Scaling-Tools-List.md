## Tool Selection Framework

**Questions to Ask**:
1. Do we need this now or can we wait?
2. What's the alternative (free/cheaper)?
3. What's the switching cost later?
4. Does it integrate with our stack?
5. Will we use 80% of features?

## Cost Management Tips

1. **Audit quarterly** - Cancel unused tools
2. **Negotiate** - Ask for startup discounts
3. **Annual plans** - Save 20-30% if you're sure
4. **Share accounts** - Within legal limits
5. **Alternatives** - Always know cheaper option
